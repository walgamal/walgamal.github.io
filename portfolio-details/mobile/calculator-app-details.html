Calculator App

Project information

  • Category: Mobile App
  • Completion Date: April 5, 2022
  • Technologies Used: Java, Android Studio, XML
  • GitHub URL: Calculator App Repo

A fully functional calculator designed to handle a range of mathematical operations. Users can input positive and negative real numbers and perform arithmetic operations such as addition, subtraction, multiplication, division, and modulus. Additionally, the app features a square root function for calculating square roots of numbers. It offers two clear options: "C" clears the entire display screen and the current calculation, while "CE" clears the most recent entry. Moreover, the app is designed to seamlessly handle device rotations while maintaining its aesthetic appeal and usability, ensuring a smooth user experience across different orientations.
