WhatToWatch

Project information

  • Category: Mobile App
  • Completion Date: April 27, 2022
  • Technologies Used: Java, Volley Library, Android Studio, Android Jetpack, JSON, OMDb API
  • GitHub URL: WhatToWatch Repo

WhatToWatch, a java mobile app, facilitates movie discovery by allowing users to search for and view detailed information about movies. Users can input a movie title through the search bar, triggering a request to an online API for movie data. Once retrieved, the app displays essential details, including the movie's title, poster, genre, plot, language, and ratings, in the display portion of the app. This seamless interaction empowers users to explore a vast array of movie options and make informed decisions on what to watch.
