Cat Facts

Project information

  • Category: Mobile App
  • Completion Date: April 22, 2022
  • Technologies Used: Java, Volley Library, Android Studio, XML, JSON, The Cat API
  • GitHub URL: Cat Facts Repo

Cat Facts is a android mobile application designed to provide users with information about different cat breeds. Users can explore a variety of cat breeds, viewing images and learning about each breed's origin and temperament. The app dynamically fetches data from an external API allowing users to select a breed from a dropdown menu and instantly see relevant information displayed on their device. With a user-friendly interface and comprehensive breed information, the app offers an engaging experience for cat enthusiasts seeking to learn more about their favorite feline companions.
